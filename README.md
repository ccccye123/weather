# 简介
这是一个方便查询天气信息的微信小程序，通过后台天气数据的处理聚合，统一返回给微信小程序界面。

# 功能
- 基本天气信息，温度、湿度、风向等
- 生活指数，穿衣、洗车、紫外线灯生活指数
- 未来的天气预告
- 多天气数据源

# 涉及技术框架
- Spring
- Redis
- Docker
- [Nacos](https://github.com/alibaba/nacos "Nacos")
- Maven
